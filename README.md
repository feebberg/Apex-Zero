# APEX ZERO SYSTEMS — DESKTOP OS

Apex Zero is a high‑contrast, desktop‑style game launcher designed to feel like a premium operating system interface.

Live site:  
**https://feebberg.github.io/Apex-Zero/**

---

## 🚀 Features

### ✔ Desktop‑Style Interface
The homepage loads into a minimal black‑ops desktop with clickable app icons.

### ✔ Boot‑Up Loading Screen
Users see a short Apex Zero Systems initialization sequence before the desktop appears.

### ✔ App Icons
Each game appears as an icon on the desktop.  
Clicking an icon opens the game inside a window.

### ✔ Windowed Game Launcher
Games open in a centered OS‑style window with:
- a title bar  
- a close button  
- an embedded iframe  

### ✔ Smooth Animations
Fade‑in transitions and loading animations create a premium feel.

## 🖱️ How to Use the Interface

1. Wait for the Apex Zero loading screen to finish.  
2. Click any icon on the desktop.  
3. A window will open containing the game.  
4. Click **✕** in the top‑right to close the game window.
